# Arduino LCD Dungeon Duel

Arduino LCD Dungeon Duel is a small reaction-based RPG combat game built with an Arduino Uno, a 16x2 I2C LCD display, buttons, and LEDs.

The player fights through multiple enemies, including a goblin, an orc, and a dragon. The LCD shows the current enemy, combat messages, and health values, while LEDs are used to display player HP and signal incoming attacks. During combat, the player must react quickly by pressing the parry button when the action LED lights up. The player can also choose different attack types using dedicated buttons.

## Features

- 16x2 I2C LCD interface
- Custom LCD pixel characters for enemies
- Goblin, orc, and dragon encounters
- HP display using LEDs
- Reaction-based parry mechanic
- Multiple attack buttons
- Difficulty selection
- Game over and win states

## Hardware Used

- Arduino Uno
- 16x2 I2C LCD display
- Push buttons
- LEDs
- Resistors
- Breadboard and jumper wires

## Project Goal

The goal of this project was to create a simple but playable Arduino game that combines basic electronics, input handling, LCD output, timing logic, and game-state management.
